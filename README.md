# 原邑销售管理平台 (YYSystem)

> 面向企业销售管理的综合业务系统，涵盖客户管理、合同订单、采购仓储、财务结算等核心业务模块。

---

## 📋 系统概述

原邑销售管理平台是一款专为企业销售团队设计的综合管理系统，通过数字化手段实现从客户开发、报价、合同签订到采购、仓储、财务结算的全流程管理。

### 核心特性

- **全流程管理**：覆盖销售、采购、仓储、财务完整业务链条
- **实时数据看板**：首页统计卡片直观展示待办事项和关键指标
- **灵活权限控制**：基于角色的细粒度权限管理
- **现代化UI设计**：采用 Indigo 主题配色，响应式布局

---

## 🏗️ 系统架构

### 技术栈

| 层级 | 技术 |

## 📦 功能模块

### 1. 工作台 (Dashboard)

**路径**：`/dashboard`

首页工作台提供全局视角的业务概览：

- **快捷入口**：方形图标卡片，快速跳转常用功能
  - 新增客户、新增报价单、新增合同
  - 费用报销、创建采购单、销售出库单

- **统计卡片**：6个待办指标实时展示
  - 待确认报价、待审核合同、待审核发票
  - 待审核报销、待审核采购、待审核入库

- **公告通知**：系统公告列表，支持管理入口

- **最新数据**：展示最新客户、报价单、合同

---

### 2. 客户管理 (CRM)

#### 2.1 客户管理
**路径**：`/crm/customer`

- 客户信息录入与维护
- 客户分类标签
- 联系人管理
- 客户公海池

#### 2.2 报价管理
**路径**：`/crm/quote`

- 报价单创建与审批
- 报价明细管理
- 报价状态跟踪（待确认/已确认/已作废）
- 统计：销售合计、成本合计、预计毛利

#### 2.3 合同订单
**路径**：`/crm/contract`

- 合同创建与审批流程
- 合同金额、收款、开票管理
- 商品出库状态跟踪
- 居间费管理
- 统计：总销售额、总收款、总应收款、已开票金额

#### 2.4 项目跟进
**路径**：`/crm/follow`

- 销售跟单记录
- 跟进状态管理
- 预算跟踪
- 统计：跟单总数、跟进客户数、本月跟单、预算总额

---

### 3. 财务管理

#### 3.1 收支记录
**路径**：`/crm/finance`

- 收入/支出记录管理
- 收支类型配置

#### 3.2 保证金管理
**路径**：`/crm/finance/deposit`

**路径**：`/crm/expense`

| 权限码 | 说明 |
|--------|------|

| `CM` | 客户管理 |

| `CO` | 合同订单 |
| `IM` | 发票管理 |
| `PA` | 采购申请 |
| `OM` | 出库管理 |
| `WHM` | 仓库管理 |

## 🚀 部署说明

### 环境要求

- **JDK**: 21+
- **Node.js**: 18+
- **MySQL**: 8.0+

### 启动命令

```bash
# 后端启动
cd yysystem-backend
mvn spring-boot:run

# 前端启动
cd yysystem-web
npm install
npm run dev
```

### 默认端口

- 前端：http://localhost:3000
- 后端：http://localhost:8080
